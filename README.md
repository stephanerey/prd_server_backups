# Server Backups PRD

Ce dépôt contient le PRD et les spécifications d'implémentation pour créer un template de sauvegarde serveur basé sur restic.

Objectif : permettre à Codex de générer un projet déployable sur des serveurs Linux, avec configuration interactive via wizard, destinations NAS agnostiques, support multi-cibles, rétention, tests de restauration et rapports email.

Point d'entrée pour Codex : lire `PRD.md` puis implémenter les PR dans l'ordre.
